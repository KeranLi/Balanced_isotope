# 氮同位素体系设计文档

基于 Kang et al. (2023) 和 Ma et al. (2025) 的氮循环箱式模型实现

## 1. 模型概述

### 1.1 双箱稳态模型

```
┌─────────────────────────────────────────────────────────────────┐
│                    大气 N₂ (δ¹⁵N = 0‰)                          │
│                         ↓ F_fix (固氮作用)                       │
│                         ε_fix = -2‰ ~ +1‰                       │
└─────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────┐              ┌─────────────────────────────┐
│ N_fixer/ammonium    │──F_remin──→│ N_assimilator/nitrate       │
│ (固氮生物-铵储库)    │   (再矿化)  │ (硝酸盐同化生物-硝酸盐储库)  │
│                     │              │                             │
│  δ¹⁵N_ammonium      │              │  δ¹⁵N_nitrate               │
│  = δ¹⁵N_atmo + ε_fix│              │  = δ¹⁵N_ammonium + ...      │
└─────────────────────┘              └─────────────────────────────┘
        │ F_fixer_burial                        │ F_assimilator_burial
        │ (固氮生物埋藏)                         │ (硝酸盐同化生物埋藏)
        ▼                                        ▼
   ┌─────────────────────────────────────────────────────────────┐
   │                    沉积物 δ¹⁵N_sed                          │
   │  = (1-f_assimilator) × δ¹⁵N_ammonium                       │
   │    + f_assimilator × δ¹⁵N_nitrate                          │
   └─────────────────────────────────────────────────────────────┘
                              ↑
         ┌────────────────────┴────────────────────┐
         │ F_wcd (水柱反硝化)  │  F_sd (沉积反硝化) │
         │ ε_wcd = -30‰~-22‰  │  ε_sd = 0‰        │
         └─────────────────────────────────────────┘
```

### 1.2 核心方程

**储库质量平衡：**

固氮生物-铵储库：
```
F_fix = F_remin + F_fixer_burial ...................... (1)
```

硝酸盐同化生物-硝酸盐储库：
```
F_remin = F_wcd + F_sd + F_assimilator_burial ......... (2)
```

**储库同位素平衡：**

固氮生物-铵储库：
```
δ¹⁵N_ammonium = δ¹⁵N_atmo + ε_fix ..................... (3)
```

硝酸盐同化生物-硝酸盐储库：
```
δ¹⁵N_nitrate = δ¹⁵N_ammonium - (F_wcd×ε_wcd + F_sd×ε_sd) / F_remin ............ (4)
```

**硝酸盐占比定义：**
```
f_assimilator = F_assimilator_burial / F_total_burial ...... (5)
```

**沉积物氮同位素：**
```
δ¹⁵N_sed = (1 - f_assimilator) × δ¹⁵N_ammonium + f_assimilator × δ¹⁵N_nitrate ...... (6)
```

## 2. 关键参数

| 参数 | 符号 | 数值范围 | 单位 | 来源 |
|------|------|---------|------|------|
| 固氮通量 | F_fix | 205 | Tg N/a | Kang et al. 2023 |
| 总埋藏通量 | F_total_burial | 25 | Tg N/a | Kang et al. 2023 |
| 沉积反硝化分馏 | ε_sd | 0 | ‰ | 文献 |
| 固氮分馏 | ε_fix | -2 ~ +1 | ‰ | 均匀分布 |
| 水柱反硝化分馏 | ε_wcd | -30 ~ -22 | ‰ | 均匀分布 |
| 大气氮同位素 | δ¹⁵N_atmo | 0 | ‰ | 标准 |

## 3. 模拟功能

### 3.1 正向模拟
从 f_assimilator 计算 δ¹⁵N_sed

### 3.2 反向模拟（反演）
从 δ¹⁵N_sed 计算 f_assimilator

注意：δ¹⁵N_sed 与 f_assimilator 存在非线性关系
- 当 f_assimilator = 0 时，δ¹⁵N_sed ≈ -0.4‰
- 当 f_assimilator ≈ 0.48 时，δ¹⁵N_sed 达到最大值 (~5.7‰)
- 当 f_assimilator = 1 时，δ¹⁵N_sed ≈ -0.7‰
